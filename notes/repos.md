## Repositories overview

During this course, we have worked with the following GitHub repositories:

- **Spring Beans**  
  [BrightBoost/beandemo](https://github.com/BrightBoost/beandemo)  
  Focus: Dependency Injection, beans, and application context basics.

- **Spring Basics**  
  [BrightBoost/yearup-spring-dec25](https://github.com/BrightBoost/yearup-spring-dec25)  
  Focus: Spring Boot fundamentals, project structure, and configuration.

- **Spring Sakila Project (JDBC)**  
  [BrightBoost/sakila-spring-example-jdbc](https://github.com/BrightBoost/sakila-spring-example-jdbc)  
  Focus: Building REST APIs with Spring Web and JDBC (no JPA).

- **Spring Security Primer**  
  [BrightBoost/spring-security-primer](https://github.com/BrightBoost/spring-security-primer)  
  Focus: Authentication, authorization, securing Spring applications
